HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages web applications. Web browsers receive HTML documents from a web sever or from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and orginally included for the appearance of the document.

HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interative forms, may be embedded into the rendered page. It provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, qoutes and other items. HTML elements are delineated by tags, written using angle brackets.
